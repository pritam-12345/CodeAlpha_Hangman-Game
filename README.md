# 🎯 Hangman Game

A clean and interactive **command-line Hangman Game** built with **Python**. The game challenges players to guess a randomly selected word one letter at a time while managing a limited number of incorrect attempts.

This project demonstrates core Python programming concepts such as loops, conditional statements, string manipulation, lists, functions, and randomization, making it an excellent beginner-friendly project.

---

## ✨ Features

- 🎲 Random word selection from a predefined word list
- 🔤 Letter-by-letter word guessing
- ✅ Input validation for user-friendly gameplay
- 🚫 Duplicate guess detection
- ❤️ Maximum of 6 incorrect attempts
- 📊 Displays remaining attempts and guessed letters
- 💻 Simple and interactive console interface
- 📚 Clean, readable, and well-structured Python code

## 🚀 Getting Started

### Prerequisites

- Python 3.x installed on your system

### Installation

1. Clone this repository

```bash
git clone https://github.com/your-username/Hangman-Game.git
```

2. Navigate to the project directory

```bash
cd Hangman-Game
```

3. Run the game

```bash
python hangman.py
```

---

## 🎮 Gameplay

- A random word is selected at the start of the game.
- Guess one letter at a time.
- Each incorrect guess reduces your remaining attempts.
- You have **6 incorrect attempts** to guess the complete word.
- Guess the word before your attempts run out to win!
